All Claude Code skills are installed globally at `~/.claude/skills/` (45 skills total).

A complete registry with reinstall script lives at `~/.claude/SKILLS-REGISTRY.md`. It contains:
- Full inventory organized by category (Design, Obsidian, Research, Dev Workflows, Context Engineering)
- Source GitHub URLs for every skill
- A single bash script that reinstalls everything from scratch

- https://github.com/kepano/obsidian-skills (5 skills)
- https://github.com/uditgoenka/autoresearch (1 skill + 10 commands)
- https://github.com/anthropics/skills (2 skills)
- https://github.com/obra/superpowers (14 skills + 3 commands + 1 agent)
- https://github.com/muratcankoylan/agent-skills-for-context-engineering (14 skills)
